## Objective
The objective of this project is to check if a string is a palindrome using a linked list.
Instead of arrays or built-in structures, each character is stored in a node of a linked list, showing how linked lists can be applied to string problems.

## Flow of the Project
1. 	Input Handling
      • 	User enters a string.
      • 	The string is normalized (converted to lowercase, spaces removed).
2. 	Linked List Creation
      • 	Each character is added as a node in the linked list.
3. 	Palindrome Check
      • 	The linked list is converted back into a string.
      • 	The string is compared with its reversed version.
4. 	Output
      • 	Display whether the string is a palindrome or not.

## Use Case
• 	Educational Purpose: Helps students understand how linked lists store and process data.
• 	Data Structure Practice: Demonstrates traversal and manipulation of linked lists.
• 	Interview Preparation: A common problem to test linked list knowledge.
• 	Real-world Applications:
• 	Checking palindromic patterns in sequences (DNA, text).
• 	Useful in algorithms where linked lists are the primary data structure.
