## Objective
The objective of this project is to check if a string is a palindrome while ignoring spaces and case differences.
This ensures that phrases like  or  are correctly identified as palindromes.

## Flow of the Project
1. 	Input Handling
      • 	User enters a string.
      • 	The string is normalized by removing spaces and converting all characters to lowercase.
2. 	Palindrome Check
      • 	Two pointers ( and ) compare characters from both ends.
      • 	If all characters match while moving inward, the string is a palindrome.
      • 	If any mismatch occurs, it is not a palindrome.
3. 	Output
      • 	Display whether the string is a palindrome or not.
